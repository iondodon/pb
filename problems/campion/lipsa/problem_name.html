problemă favorită 

lipsa


Timp maxim de execuţie / test:
0.6s
Memorie totala disponibilă / stivă:
2MB / 1MB

Pe Permutation, de unde Excelența Sa, plenipotențiarul ambasador !P provine, există o permutare a numerelor naturale {1, ..., N}, fără unul dintre ele. Pentru a-și îmbogăți universul, el vrea să știe care este acel număr lipsă.

Cerinţă

Realizează un program care să determine numărul lipsă.

Date de intrare

Programul citește din fișierul de intrare lipsa.in. Pe prima linie se găsește N, urmat de cele N-1 numere naturale distincte.

Date de ieşire

Programul scrie pe prima linie din fișierul de ieșire lipsa.out numărul lipsă. Numărul natural scris este urmat de caracterul sfărșit de linie.

Restricţii

1 < N < 1000001

Exemple

lipsa.inlipsa.outExplicaţii
5 3 1 5 4 2 Numărul N este 5. Numărul lipsă este 2.
5 1 2 3 5 4 Numărul N este 5. Numărul lipsă este 4.
