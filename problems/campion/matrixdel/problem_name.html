matrixdel


Timp maxim de execuţie/test:
0.1 secunde
Memorie totală disponibilă/stivă:
4MB/1 MB

Fie o matrice de numere naturale cu L linii numerotate de la 1 la L şi C coloane numerotate de la 1 la C. Din această matrice se elimină P linii şi Q coloane.

Cerinţă

Să se afişeze matricea rămasă după ştergerea liniilor şi coloanelor.

Date de intrare

Fişierul matrixdel.in conţine pe prima linie numerele L şi C. Pe următoarele L linii se află câte C numere naturale separate prin spaţii reprezentând câte o linie din matrice. Pe linia L+2 se află numărul natural P, iar pe linia L+3 se găsesc, în ordine strict crescătoare şi separate prin spaţii, P numere naturale i1, i2, ..., iP reprezentând indicii liniilor care se vor şterge din matrice. Pe linia L+4 se află numărul natural Q, iar pe linia L+5 se găsesc, în ordine strict crescătoare şi separate prin spaţii, Q numere naturale j1, j2, ..., jQ reprezentând indicii coloanelor care se vor şterge din matrice.

Date de ieşire

Fişierul matrixdel.out va conţine L–P linii. Pe fiecare linie se vor afla, separate prin spaţiu, cele C–Q elemente din coloanele rămase.

Restricţii

  • 3 <= L, C <= 100
  • 1 <= P < L; 1 <= Q < C
  • 1 <= i1 < i2 < ... < iP <= L
  • 1 <= j1 < j2 < ... < jQ <= C
  • elementele matricei sunt numere naturale cuprinse între 0 şi 100
